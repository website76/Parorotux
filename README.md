## Features
- **Live Predictions**: Real-time, API-driven predictions with calculated insights.
- **User Levels**: Progress through levels by meeting specific milestones.
- **API-Driven Calculations**: Ensures accurate and reliable results.
- **User-Friendly Interface**: Designed for easy navigation and gameplay.

## How to Play

1. **Interface Overview**:
   - **Period**: Displays the current prediction period.
   - **Result**: Shows the outcome after making a prediction.
2. **Make Predictions**: Click the **Predict** button to submit for the period.
3. **View Results**: The **Result** section will update based on API calculations.
4. **Maintain Levels**: Consistent gameplay helps you maintain your level.
